Basis

xorg server and xinit for startx.

1

sudo pacman -S xorg xorg-xinit

Enable proxy (optional)

1

export ALL_PROXY=socks5://192.168.15.136:7890

Clone src.

1
2
3

git clone https://friedrich-top.org/dwm
git clone https://friedrich-top.org/slstatus
git clone https://friedrich-top.org/slock

Build src and install.

1
2

cd ./dwm
sudo make clean install

Prepare a terminal emulator.

1

sudo pacman -S alacritty

Copy .xinitrc to HOME directory

1

cp /etc/X11/xinit/xinitrc ~/.xinitrc

Edit .xinitrc

1
2

... &
exec dwm

Run dwm from TTY

1

startx

Autostart programs with dwm

Add autostart patch

1

git clone https://dwm.suckless.org/patches/autostart/dwm-autostart-20210120-cb3f58a.diff
1

patch < patches/dwm-autostart-20210120-cb3f58a.diff
1

sudo make clean install

Edit autostart script

1
2

mkdir ~/.dwm
vim ~/.dwm/autostart.sh
1
2
3

#!/bin/bash
... &
picom # example
